How to run the project
1 - Download .Net 5.0 from https://dotnet.microsoft.com/download
2 - Load the project folder to Visual Studio Code (Or any IDE)
3 - From the IDE terminal, type "dotnet run" and hit Enter
Alternative: The project has already been built, meaning the application can be run from the .exe file located in the bin folder
Extra: To reset the application directory configuration, locate and delete EDApp.dll.config in the bin folder
